A canvas based class implementing a popupmenu with rounded corners.


this class make use of HierPop classes which are

Copyright by Noah Desch <http://www.wireframesoftware.com/



Properties


ListIndex as integer

Get or set the currently selected menu item.


Menu as HCMenu

The associated menu.



Methods


AddRow(Item as string)

Add a menu item with the passed text.


AddRows(Items() as string)

Add a group of items passed as string array.


AddSeparator()

Add a separator line.


DeleteAllRows()

Remove all the menu items.


InsertRow(RowNumber as integer, Item as string)

Insert a menu item in the position specified with RowNumber.


List(Index as integer) as string

Return the text of menu item at Index.


ListCount() as integer

Return the number of menu items.


RemoveRow(Index as integer)

Remove the menu item at Index.


Text() as String

Return the currently selected item's text.


AddRows(Items() as string)

Add a group of items passed as string array.


Events


Change()

This event is fired when the selected menu item is changed, either via user selection or setting the ListIndex property.


To handle an associated menu, see the example or the HierPop documentation.
